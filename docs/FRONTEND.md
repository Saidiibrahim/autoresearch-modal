# Frontend Scope

This repository has no first-party frontend surface.

## Current State

- No web app, dashboard, or design system ships from this repo.
- User interaction is command-driven through Modal CLI and repo-local documentation.

## If That Changes

- Add a product spec under `docs/product-specs`.
- Add design documentation under `docs/design-docs/`.
- Define UI validation and accessibility expectations in `docs/QUALITY_SCORE.md` and `docs/RELIABILITY.md`.
